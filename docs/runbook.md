# Runbook

Practical steps for running and operating the project.

## Local setup

- Install project dependencies.
- Copy `.env.example` to `.env` and set local values.
- Start the app with the project-specific command.

## Common operations

- Run tests.
- Run lint/format checks.
- Build a release artifact.

## Incident notes

- Capture symptoms.
- Capture recent changes.
- Capture rollback/mitigation steps.
